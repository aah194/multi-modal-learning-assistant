# Multi-Modal Learning Assistant

A Streamlit-based AI learning assistant powered by Google's Gemini API.

## Features

* PDF Question Answering (RAG)
* Multi-PDF Knowledge Base
* Conversational Memory
* Image Understanding
* PDF + Image + Text Reasoning
* Voice Assistant
* Text-to-Speech Support
* FAISS Vector Search
* Sentence Transformers Embeddings

## Tech Stack

* Python
* Streamlit
* Gemini 2.5 Flash
* FAISS
* Sentence Transformers
* PyPDF2
* SpeechRecognition
* pyttsx3
* Pillow

## Setup

1. Clone repository

```bash
git clone <repository-url>
```

2. Create virtual environment

```bash
python -m venv venv
```

3. Install requirements

```bash
pip install -r requirements.txt
```

4. Create .env file

```text
GEMINI_API_KEY=your_api_key
```

5. Run application

```bash
streamlit run app.py
```
